# ``Swift/String/LocalizationValue/StringInterpolation/appendInterpolation(_:format:)-7jfr8``

@Metadata {
    @DocumentationExtension(mergeBehavior: override)

Appends a formatted attributed string to a string interpolation.

Don’t call this method directly; the compiler uses it when interpreting string interpolations.

- Parameters:
  - value: A formattable value to append.
  - format: A `FormatStyle` to format `value` to a Foundation `AttributedString`, prior to appending.
