# ``Swift/String/LocalizationValue/StringInterpolation/appendInterpolation(_:options:)``

@Metadata {
    @DocumentationExtension(mergeBehavior: override)
}

Appends an attributed string to a string interpolation, applying the specified options.

Don’t call this method directly; the compiler uses it when interpreting string interpolations.

- Parameters:
  - attrStr: An `AttributedString` to append.
  - options: An `AttributedString.InterpolationOptions` instance to customize the interpolation behavior.
