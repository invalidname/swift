# ``Swift/String/LocalizationValue/StringInterpolation/appendInterpolation(_:format:)-3xioh``

@Metadata {
    @DocumentationExtension(mergeBehavior: override)
}

Appends a formatted collection of strings that perform an out-of-process localization to a string interpolation.

Don’t call this method directly; the compiler uses it when interpreting string interpolations.

- Parameters:
  - value: The localizable string to append.
  - format: A `ListFormatStyle` to apply to the collection.
