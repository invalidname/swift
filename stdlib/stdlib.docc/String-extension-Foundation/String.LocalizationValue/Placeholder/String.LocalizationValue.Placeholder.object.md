# ``Swift/String/LocalizationValue/Placeholder/object``

@Metadata {
    @DocumentationExtension(mergeBehavior: override)
}

The object type, as used for replacement values with the localized string placeholder syntax.

To insert a object into a placeholder, use the syntax `\(placeholder: .object)`.

``Swift/String/LocalizationValue`` supports interpolating `NSObject` instances and Foundation types that bridge to `NSObject` types.
