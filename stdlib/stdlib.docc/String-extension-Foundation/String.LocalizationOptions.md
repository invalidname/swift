# ``Swift/String/LocalizationOptions``

@Metadata {
    @DocumentationExtension(mergeBehavior: override)
}

Options to apply when initializing a localized string.

Use this type to configure how the system localizes strings. You can use the ``Swift/String/LocalizationOptions/replacements`` property to provide replacement values for localizable strings that use the `\(placeholder:)` syntax.
