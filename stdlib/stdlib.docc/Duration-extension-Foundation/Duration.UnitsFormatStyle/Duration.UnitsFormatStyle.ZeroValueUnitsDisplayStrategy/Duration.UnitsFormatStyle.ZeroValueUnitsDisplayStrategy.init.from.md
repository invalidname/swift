# ``Swift/Duration/UnitsFormatStyle/ZeroValueUnitsDisplayStrategy/init(from:)``

@Metadata {
    @DocumentationExtension(mergeBehavior: override)
}

Creates a new instance by decoding from the given decoder.

This initializer throws an error if reading from the decoder fails, or if the data read is corrupt or otherwise invalid.

- Parameter decoder: The locale to use when formatting a duration.
